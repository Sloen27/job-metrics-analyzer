# Job Metrics Analyzer

Анализатор метрик выполнения джоб из CSV логов.

## Описание

Веб-приложение для загрузки и анализа CSV файлов с логами выполнения фоновых задач (jobs). Позволяет сравнивать метрики производительности между периодами времени и отслеживать динамику показателей.

## Функции

- **Загрузка CSV файлов** - перетаскивание или выбор файла
- **Выбор джобы через вкладки** - каждая джоба отображается как отдельная вкладка
- **Гибкий выбор интервалов** - сравнение периодов от 3 часов до недели
- **Сравнение метрик** - текущий период vs предыдущий с индикаторами трендов
- **Визуализация данных** - графики динамики по часам и дням
- **Детальная таблица** - метрики по дням с индикаторами изменений

## Интервалы сравнения

- Последние 3 часа vs предыдущие 3 часа
- Последние 6 часов vs предыдущие 6 часов
- Последние 12 часов vs предыдущие 12 часов
- Последние 24 часа vs предыдущие 24 часа
- Сегодня vs вчера (эквивалентные периоды)
- Последние 7 дней vs предыдущие 7 дней
- Эта неделя vs прошлая неделя

## Цветовая индикация

- 🟢 **Зелёный** - улучшение показателей
- 🟡 **Жёлтый** - деградация (требует внимания)
- ⚪ **Серый** - стабильность (изменения < 5%)

## Формат CSV файла

Ожидается CSV файл с разделителем `;` и следующими колонками:

| Колонка | Описание |
|---------|----------|
| Id | Уникальный идентификатор запуска |
| Name | Название джобы |
| StartDate | Дата/время начала (ISO формат) |
| EndDate | Дата/время окончания (ISO формат) |
| Status | Статус (1=успех, 3=ошибка) |
| Error | Текст ошибки (если есть) |
| Metrics | Текстовое описание метрик |

## Технологии

- **Next.js 16** - React фреймворк с App Router
- **TypeScript** - типизация
- **Tailwind CSS** - стилизация
- **shadcn/ui** - UI компоненты
- **Zustand** - управление состоянием
- **Recharts** - графики
- **date-fns** - работа с датами

## Запуск

```bash
# Установка зависимостей
bun install

# Режим разработки
bun run dev

# Продакшн сборка
bun run build
bun run start
```

## Скриншоты

Приложение предоставляет:

- Таблицу метрик по дням
